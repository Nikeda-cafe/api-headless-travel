# Headless Travel api連携用アプリケーション
※現在はお知らせ情報の管理とAPI配信、お問い合わせメールの送信機能を実装している。

## 使用技術
* Express.js
* Nginx
* MariaDB
* webpack5
* JWT認証
* Bootstrap4
* AWS Lightsail
* AWS CloudFront
